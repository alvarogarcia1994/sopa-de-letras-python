# Spanish (ES)
# Juego de sopa de letras (Python)
- Juego de sopa de letras para terminal escrito en Python.
  El jugador debe encontrar todas las palabras ocultas generadas al azar en el tablero.

## Características

- El tablero de juego de sopa de letras se genera aleatoriamente
- Las palabras se colocan en **8 direcciones posibles (horizontal, vertical, diagonal y en sentido inverso)**
- La lista de palabras externa se carga desde un fichero de texto plano.
- Validación de entrda por teclado 
- Opción de volver a jugar al finalizar la partida

# English (EN)
# Word Search Game (Python)

- A terminal-based word search game written in Python.
  The player must find all hidden words in a randomly generated board.

## Features

- Randomly generated word search board
- Words placed in **8 possible directions**
- Horizontal, vertical and diagonal
- Words can appear reversed
- External word list loaded from file
- Input validation and game restart option
- Clean and modular code structure
